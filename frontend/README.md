# Retrievex Frontend

Phase 1 frontend scope is limited to:

- Dashboard.
- New Retrieval Request.
- Provider Analytics.

The frontend must show active retrieval requests, escrow status, recent retrieval history, Storage Provider performance summary, retrieval request creation, and provider analytics.

No marketing-heavy UI or non-PRD features belong here.

Framework: React with Vite.

Reason: minimal static dashboard build suitable for Vercel or static hosting, matching the PRD infrastructure scope.
